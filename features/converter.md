# DeluxeMenus Converter

Sheets can automatically detect and convert DeluxeMenus configuration files into its native format.

### Automatic Detection

When a DeluxeMenus file is loaded, Sheets will:
1.  Detect that the file is in the DeluxeMenus format.
2.  Convert it into the Sheets format.
3.  Automatically save the converted format back to the file.

### How to Convert

Simply place your DeluxeMenus files into the `plugins/Sheets/menus/` folder and run `/sheets reload`.

### Supported Conversions

*   Menu settings (title, size, inventory type).
*   Buttons and items.
*   [Requirements](requirements.md) (view and click).
*   [Actions](actions.md) (player and console commands).
*   Placeholder syntax conversion.

### Why Use the Converter?

*   **Ease of Transition**: Move your existing menus from DeluxeMenus to Sheets with minimal effort.
*   **Unified Support**: Take advantage of Sheets' Bedrock form support for your existing menus.
*   **Modern Features**: Use new features like Chest Edit Mode and Anvil input with your converted menus.
